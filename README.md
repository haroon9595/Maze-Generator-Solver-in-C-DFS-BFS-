# Maze Generator & Solver in C# (DFS & BFS)

A professional **Maze Generator and Solver** developed in **C#** using **Windows Forms**. This project demonstrates **Data Structures and Algorithms (DSA)** concepts through interactive visualization. The maze is randomly generated using **Depth-First Search (DFS)** and solved using **Breadth-First Search (BFS)** to find the shortest path.

---

## Features

- Random maze generation using DFS
- Maze solving using BFS (shortest path guaranteed)
- Step-by-step animated visualization of algorithms
- Multiple maze sizes and configurable start/end points
- Speed control for animation
- Color theme selection for visualization
- Real-time score/path length display
- Reset and regenerate maze options
- Clean, professional GUI using Windows Forms



## Technologies and Techniques

- **Programming Language:** C#
- **Framework:** Windows Forms
- **Algorithms:** DFS (for maze generation), BFS (for shortest path)
- **Data Structures:** Stack, Queue, 2D Array, Graph representation

---

## How It Works

1. **Maze Generation (DFS)**
   - Maze represented as a grid with walls
   - DFS carves paths by removing walls recursively
   - Backtracking ensures a perfect maze (one unique path)

2. **Maze Solving (BFS)**
   - BFS explores neighboring cells level-by-level
   - Parent information stored for path reconstruction
   - Guarantees shortest path visualization

3. **Visualization**
   - Current cell, visited cells, walls, and final path highlighted with colors
   - Animation with controlled speed
